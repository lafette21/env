## Prerequisites

### Arch

Installation via `pacman`

#### fzf

```
pacman -Sy fzf
```

#### zsh-autosuggestions

```
pacman -Sy zsh-autosuggestions
```

#### zsh-syntax-highlighting

```
pacman -Sy zsh-syntax-highlighting
```

### macOS

Installation via `brew`

#### fzf

```
brew install fzf
```

#### zsh-autosuggestions

```
brew install zsh-autosuggestions
```

#### zsh-syntax-highlighting

```
brew install zsh-syntax-highlighting
```
